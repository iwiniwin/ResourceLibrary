Package Validation Suite

[com.unity.package-validation-suite][] 预览

描述

Unity Package Manager 验证套件，用于在考虑发布某个包之前评估该包的完整性。目前，这是一个 Unity 内部工具，不提供该包的相关支持。

版本信息

适用于 Unity 的实验性或预览功能

以下包版本可能无法安全地在实际生产中与 Unity 2019.4 版本搭配使用：
