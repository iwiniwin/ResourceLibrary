删除资源包草案

如果要废弃资源包草案并重新开始，则可以删除草案：

1.打开 Publisher Administration 页面，然后选择 Packages 选项卡。

2.从 Create New Package 按钮旁边的下拉列表中选择 Draft。

此时会显示草案资源包列表。

![草案资源包列表](../uploads/Main/AssetStoreDeletePkg.png)

3.单击要删除的草案的名称。

此时会显示 **Create new package** 页面。该页面显示资源包的详细信息。

4.向下滚动到页面底部，然后单击 Delete Draft 按钮。

![Delete Draft 按钮位于资源包详细信息页面的底部](../uploads/Main/AssetStoreDeletePkg-delbtn.png)

5.在对话框中单击 OK 以确认删除，或单击 Cancel 以保留草案。
