Unity VR 开发入门

要开始 VR 开发，Unity 建议使用 XR Management 来加载和管理目标平台 SDK。有关如何使用 XR 插件管理系统来配置项目的说明，请参阅针对 XR 配置 Unity 项目页面。

第三方插件

Oculus

  • 如果是进行 Oculus Quest 开发，则需要为 Android 构建目标安装和加载 Oculus XR 插件。
  • 如果是进行 Oculus Rift 和 Rift S 开发，则需要为 Windows 构建目标安装和加载 Oculus XR 插件。
  • 有关更多信息，请参阅 Oculus 开发者文档。

Windows Mixed Reality

  • 如果是进行 Windows MR VR 头盔开发，则需要安装和加载适用于通用 Windows 平台的 Windows XR 插件，并使用 Microsoft 的 Mixed Reality Toolkit (MRTK 2.3) 来添加混合现实输入和交互。
  • 有关更多信息，请参阅 Microsoft 的 Windows Mixed Reality 文档。

Google VR

请参阅 Google 的 VR SDK 快速入门指南以了解有关 Google Daydream 和 Cardboard 的最新信息。有关 Google Cardboard Unity XR 插件的更多信息，请参阅 Google Cardboard for Unity 快速入门。

OpenVR

Valve 已经发布了他们的 OpenVR Unity XR 插件（Beta 版）。请参阅 steamvr_unity_plugin GitHub 代码仓库以查看相关文档和安装说明。

输入和交互

Unity 的 XR Interaction Toolkit 可让您为自己的 VR 应用程序增加交互性，而不必从头开始编写交互代码。此包可与 Unity 支持的 XR 插件一起使用，并通过 Package Manager 作为 Unity 2019.3 的预览包提供。
