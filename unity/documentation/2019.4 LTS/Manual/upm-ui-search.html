搜索框

您可以使用 Package Manager 窗口顶部的搜索框查找与您输入的文本匹配的所有包。在您输入文本时，Package Manager 将更新包列表，显示当前范围内匹配的每个包。例如，如果您在搜索框中输入 web，列表将显示不同的结果，具体取决于您选择的范围是 Built-in packages 还是 All packages：

可以按照包的名称或版本号查找包。Package Manager 会分析您输入的文本以检测所使用的方法，并动态更新包列表。

注意：您可以搜索 Unity 包或资源包。选择的筛选条件定义了 Package Manager 进行搜索的位置：

筛选条件设置： Package Manager 进行搜索的位置：
All packages Unity Package Manager 注册表上、范围注册表中以及项目中任何位置（包括 Packages 文件夹下）的任何包。
In Project 当前项目中已安装或嵌入的任何包（包括本地安装的包）。
In Development 在当前项目的 Packages 文件夹中。
My Assets 您帐户下已购买或下载的资源列表（位于 My Assets 页面）。
Built-in packages 任何内置包。

包名称

您可以搜索显示名称（也就是 Package Manager 窗口中显示的名称），对于 Unity 包，则可以搜索已注册包名称（以 com.unity 开头）。

搜索文本可以与包名称的开头、中间或结尾匹配。

版本号

按照版本顺序 (MAJOR.MINOR.PATCH) 输入版本号。例如，键入 1 将返回版本以 1 开头的所有包（例如，1.0.0、1.3.2、1.10.1 和 10.0.0 全都匹配，但 0.1.0 不匹配）。
