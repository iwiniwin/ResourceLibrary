Unity Test Framework

Unity Test Framework 包（以前称为“Unity Test Runner”）是一款 Unity 工具，可在__编辑__模式和__运行__模式下以及在目标平台（如独立平台、Android 或 iOS）上测试您的代码。

从 Unity 2019.2 版开始，该工具已在 Unity Test Framework 包中提供，您可以通过 Unity Package Manager 下载这个包。

有关可用版本的更多信息，请参阅 com.unity.test-framework 页面。
