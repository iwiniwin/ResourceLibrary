查找包

根据选择的范围，您可以使用搜索框搜索 Unity 包或资源包。有关更多信息，请参阅搜索框。

查找某个 Unity 包的特定版本

要查找某个特定的包版本，请执行以下操作：

1.在 Unity 中，打开 Package Manager 窗口（导航到 Unity 的主菜单，然后选择 Window > Package Manager）。

2.如果要寻找预览包，请从 Advanced 下拉菜单中选择 Show preview packages。

![Show preview packages](../uploads/Main/upm-docs.png)

3.从包的列表视图（左侧）中选择要获取信息的包。

4.在列表中展开包左侧的箭头。

此时将出现一个新行，显示 **See all versions** 链接。

![See all versions](../uploads/Main/upm-docs1.png)

5.单击 See all versions 链接。

此时将显示一个可滚动列表，其中包含所有可用版本。

![所有可用版本](../uploads/Main/upm-docs2.png)

6.单击另一个版本以将其选中。

包详细信息将显示在右侧面板中。

![选择一个版本以查看其详细信息](../uploads/Main/upm-docs3.png)

7.请注意，此示例中不再显示 verified 标签，因为 TextMesh Pro 包版本 1.2.4 未针对此示例中使用的 Unity 版本进行验证。

8.选择特定版本时，可以执行多种操作：

* 可以查看[此包是否依赖于](upm-ui-list.html#VersionList)其他任何包，包括特定的包名称和版本。
* 可以单击 **View documentation** 链接以打开与该版本匹配的[文档](upm-docs.html)。
* 可以单击 **View changelog** 链接以查看在创建包之后直至此版本期间进行的更改的列表。
* 可以单击 **View licenses** 链接以查看此包专门针对此版本的任何许可证信息。
* 如果还没有安装此包，则可以[安装](upm-ui-install.html)此版本。
* 如果已经安装了此包的另一个版本，则可以将此包[更新](upm-ui-update.html)到此版本。
* 如果以前已经安装了此包，但是不想再使用，则可以[删除](upm-ui-remove.html)此包。
* 如果此包中包含示例资源，则可以单击 **Import into project** 按钮以将其副本保存在项目的 `Asset` 文件夹下面。
