导入资源包

可在 Package Manager 窗口中导入资源包：

1.打开 Package Manager 窗口。

2.从包范围下拉菜单中，选择 My Assets。

![将范围切换为 **My Assets**](../uploads/Main/upm-ui-myassets.png)

列表视图会显示您购买的所有资源包。

3.如果 My Assets 列表中出现大量资源包，则可以按名称或版本号来搜索特定资源包。

4.从包列表中选择要导入的资源包。资源包信息将显示在详细信息视图中。

5.单击 Download 按钮（如果已启用此按钮）。如果您以前从未将这个资源包下载到此计算机或设备，则会禁用 Import 按钮，您需要先下载这个资源包。

![详细信息视图右下角的 **Download** 按钮](../uploads/Main/upm-ui-import.png)

在资源包下载到计算机的过程中，__Import__ 按钮旁边将出现进度条。__Cancel__ 按钮将取代您单击的 __Download__ 按钮。

![下载期间的进度](../uploads/Main/upm-ui-download.png)

下载完成后，__Import__ 按钮变为启用状态，__Download__ 按钮变为禁用状态。

6.单击 Import 按钮以导入选定的资源包。

Import Unity Package 窗口将显示包中已选择的所有项，然后可以直接安装。

![全新安装 Import Unity Package 对话框](../uploads/Main/CustomPackageInstallDialog.png)

7.取消选择不想导入的所有项，然后单击 Import。

Unity 会将导入的资源包的内容放入 `Assets` 文件夹中，以便您可以从 Project 窗口中进行访问。



  • 在 Unity 2019.3 中添加了 My Assets 筛选条件 NewIn20193
