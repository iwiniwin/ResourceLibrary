Learn 屏幕

通过 Unity Hub 中的 Learn 屏幕，可访问各种教程和学习资源（包括可直接导入 Unity 的示例项目），从而帮助您快速上手 Unity。

Learn 屏幕包含两个选项卡：

  • Projects
  • Tutorials

Projects

此选项卡提供了可以导入 Unity 的示例项目。每个项目都附带一个教程以及自行完成项目所需的所有资源。

要了解有关示例项目的更多信息，请单击项目的名称。

要将项目下载并导入到 Unity 中，请单击 Download Project。Unity 会为所选的示例项目下载所有必需的资源。下载完成后，单击 Open Project 即可在 Editor 中启动项目。

要访问随附的教程，请单击 View Tutorials。还可在 Unity Learn 网站上找到每个项目的视频教程。

Tutorials

此选项卡中提供的这些教程将介绍与 Unity 的界面和主要开发工具交互的基础知识。

要了解有关教程的更多信息，请单击教程的名称。

要访问教程，请单击 View Tutorial。
