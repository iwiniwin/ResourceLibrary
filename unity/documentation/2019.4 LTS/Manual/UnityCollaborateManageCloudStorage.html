管理云存储

Collaborate 服务使用云存储在云端备份整个项目，使其可在任何地方访问。

本主题介绍：

  • 确定项目当前使用的存储空间
  • 添加更多存储空间
  • 云存储数据保留
  • 将项目存档
  • 为 Collaborate 存档项目

确定项目当前使用的存储空间

云存储是基于每个组织计算的。Unity 通过汇总组织的所有 Collaborate 项目的最新版本的大小来计算组织正在使用的云存储量。只有最新版本的项目会计入存储限制。项目的历史（无头）版本不计入云存储。

要查看当前的存储使用量，请执行以下操作：

1.登录 Unity 开发者控制面板 (Developer Dashboard)。 2.在组织中选择要查看存储空间的项目。 3.在左侧导航窗口中，单击 Settings__，然后单击 Usage__。

在项目主窗口中会显示一些磁贴：

  • Cloud storage - 当前项目与组织中其他项目相比使用的存储空间以及整个云存储空间。
  • Percent of storage per project - 组织中每个项目使用的存储空间。
  • Storage used by <Project name> over time - 当前项目自创建以来一直到当前时间使用的存储空间。

添加更多存储空间

您可以通过两种方式获得更多云存储：

  • 升级到 Unity Teams Advanced，此版本也提供 25GB 的云存储。
  • 在 Unity ID 控制面板上购买存储包。每个存储包为组织增加 25GB 的云存储。

升级到 Unity Teams Advanced

要升级到 Unity Teams Advanced，请访问 Unity Teams 页面 unity3d.com/teams。在 Unity Teams 页面上，可以了解有关 Unity Teams Advanced 功能的更多信息，然后进行升级。

购买存储包

要添加存储包，请执行以下操作：

1.登录 Unity ID 控制面板。 2.在左侧导航栏中，单击 Organizations。 3.在 Organizations 主窗口中，选择一个组织。 4.在左侧导航栏中，单击 Service Usage。 5.在 Usage overview 窗口中，单击 Manage storage 按钮。 6.在 Add cloud storage 对话框中，选择所需的存储量，然后单击 Continue to purchase 按钮。如果只有一个组织，请继续执行步骤 6。 7.在 Checkout 窗口中，输入付款信息，然后单击 Pay now。

修改您签约的存储量

1.登录 Unity ID 控制面板。 2.在左侧导航窗口中，单击 Organizations。 3.在 Organizations 主窗口中，选择一个组织。 4.在左侧导航栏中，单击 Service Usage。Service Usage 窗口会显示您的合同期限以及您签约的存储包数量。 5.在 Usage overview 窗口中，单击 Manage storage 按钮。 6.在 Add cloud storage 选项卡上，输入要添加的存储包数量。 7.单击 Next 按钮。 8.在 Review order 窗口中，查看订单信息，然后单击 Confirm purchase 按钮。

可在 Manage subscription 选项卡上减少存储包数量：

  • 要删除所有签约的存储包，请关闭 Automatic Renewal。关闭自动续订后，将在续订日期删除所有存储包。
  • 要减少续订的存储包数量，请在 Select the number of storage packs you want to renew upon renewal 字段中输入要续订的存储包数量。

然后，单击 Save 按钮。

云存储数据保留

如果使用的是 Unity Teams Advanced，则 Unity 不会删除项目的任何版本历史记录。

如果使用的是 Unity Teams Basic，Unity 会将您的数据存储在云存储中至少 1 年。Unity 保留删除超过 1 年的项目历史记录的权利。您始终可以访问最新版本的项目，但只能访问过去 90 天的版本历史记录。如果使用的是 Unity Teams Basic 并希望完整访问过去 1 年的历史记录，则必须升级到 Unity Teams Advanced。升级到 Unity Teams Advanced 后，就能访问完整的历史记录。

减少云存储使用量

有三种方法可以减少云存储使用量：

  • 删除项目中的资源或减小资源大小。请务必注意，项目历史记录不计入云存储使用量。
  • 将未使用的项目存档。请参阅将项目存档。
  • 仅为 Collaborate 存档项目。仅为 Collaborate 存档项目可以释放云存储空间，同时仍然可以使用其他 Unity 服务，如 Ads、Analytics 和 Performance Reporting。请参阅为 Collaborate 存档项目。

将项目存档

将项目存档会自动将与项目关联的任何 Collaborate 云存储进行存档，且不再可以访问 Unity 服务。

要对项目存档，请执行以下操作：

1.登录 Unity 开发者控制面板 (Developer Dashboard)。 2.选择要存档的项目。 3.在左侧导航栏中，单击 Settings__，然后单击 General。 4.在主窗口的底部，阅读重要说明，如果您确定要将项目存档，请单击 Archive my Project for ALL Unity services__ 按钮。 5.在 Are you sure you want to archive this Project 对话框中，输入项目名称，然后单击 Yes, archive it 按钮。

要对项目取消存档，请执行以下操作：

1.登录 Unity 开发者控制面板 (Developer Dashboard)。 2.在主窗口右上角的 CREATE NEW PROJECT 按钮下，单击 View Archived Projects 链接。 3.找到要重新激活的项目，单击项目名称右侧的垂直省略号，然后选择 Unarchive Project。

为 Collaborate 存档项目

要仅为 Collaborate 存档项目，请执行以下操作：

1.登录 Unity 开发者控制面板 (Developer Dashboard)。 2.选择要存档的项目。 3.在左侧导航窗口中，单击 Collaborate__，然后单击 Storage。 4.在主窗口中，单击 ARCHIVE__ 按钮。

要为 Collaborate 取消项目存档，请执行以下操作：

1.登录 Unity 开发者控制面板 (Developer Dashboard)。 2.选择要取消存档的项目。 3.在左侧导航窗口中，单击 Collaborate__，然后单击 Storage。 4.在主窗口中，单击 UNARCHIVE__ 按钮。
