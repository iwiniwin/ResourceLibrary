Unity 2019.4 的系统要求

Unity Editor 系统要求

此部分列出了运行 Unity Editor 的最低要求。实际性能和渲染质量可能会因项目的复杂性而异。

最低要求 Windows macOS Linux（在预览中支持）
操作系统版本 Windows 7 (SP1+) 和 Windows 10，仅限 64 位版本。 Sierra 10.12.6+ Ubuntu 16.04、Ubuntu 18.04 和 CentOS 7
CPU X64 架构（支持 SSE2 指令集） X64 架构（支持 SSE2 指令集） X64 架构（支持 SSE2 指令集）
图形 API DX10、DX11 和 DX12 兼容型 GPU Metal 兼容型 Intel 和 AMD GPU OpenGL 3.2+ 或 Vulkan 兼容型 Nvidia 和 AMD GPU

XR 平台系统要求

在使用 XR Management 包 向 Unity XR 插件架构迁移后的 Unity 2019.3 和更高版本中，将弃用内置平台包。

这些实现方案仍然可以运行，并在 Unity Package Manager 中显示，但存在的错误和问题可能不会优先得到处理，并且在将来的发行版中将删除该包。

有关 Unity 2019.3+ 中的 Google Daydream/Cardboard、OpenVR 和 Vuforia 支持选项，请参阅 XR 插件架构页面。
