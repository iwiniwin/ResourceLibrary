许可证和激活
离线/手动激活

在线激活

要使用 Unity，需要一个激活的许可证。要查看可用的许可证类型，请访问 Unity Store。

如果已经安装了 Unity Editor，则可以按照管理许可证文档页面中的说明激活新许可证。

如果已购买了许可证但尚未使用 Unity，则可以通过 Unity Hub（可以从下载 Unity (Download Unity) 网页下载）激活许可证。

__注意：__要激活 Unity Plus 或 Pro 许可证，Hub 会要求提供许可证和序列号信息。如果购买了一个许可证，Unity 将通过电子邮件将许可证信息和序列号发送给购买者。

如果您不知道自己的 Unity 许可证信息，请与许可证的所有者联系。所有者可以通过组织 (Organization) 为您分配一个席位，或者您也可以联系 Unity 客户服务部。

激活许可证

打开 Unity Hub，然后通过窗口右上方的帐户图标登录您的 Unity ID。如果您还没有 Unity 帐户，可以访问 Unity ID 网站创建一个。登录后，单击右上角的齿轮图标，然后导航到 License Management 选项卡。

单击 Activate New License__，此时将显示一些选项，供您选择要激活的许可证的类型（__Unity Personal、__Unity Plus or Pro__）。

要激活 Unity Personal 许可证，请选择 Unity Personal 选项，然后选择最能描述您的当前状况的任一选项，然后单击 Done。

要激活 Unity Plus 或 Pro 许可证，请选择 Unity Plus or Pro 选项，然后输入序列号。

序列号密钥将发送到与您的 Unity ID 关联的电子邮件地址。必须严格按照电子邮件中显示的格式输入密钥。密钥的格式如下所示：

PS/SC/E3-XXXX-XXXX-XXXX-XXXX

注意：__要激活教育版许可证，请选择 Unity Plus or Pro__ 选项。

许可证成功激活后，Unity 会显示“Your license was activated successfully”消息。请单击 Start Using Unity 以继续，然后单击 Preferences 以创建新项目或者加载现有项目。

故障排除

故障排除

如果在激活过程中遇到任何错误代码或其他问题，可能是由以下原因引起的：

  • 您的计算机可能无法访问互联网。在这种情况下，可能需要手动激活许可证。可以在手动激活 (Manual Activation) 文档中找到有关具体操作的说明。
  • 您的防火墙/代理或互联网安全设置可能阻止 Unity 发送和接收有关许可证文件的数据。
  • 您可能在计算机上不具有正确的读/写权限来保存许可证文件。要解决此问题，可能需要将这些权限授予在尝试激活许可证时所使用的用户帐户。这是工作场所和学校特别常见的问题；请联系 IT 管理员，了解是否存在可能阻止激活过程正常进行的限制。
  • 重大的操作系统更改或更新可能会影响您的计算机 ID，这会导致许可证停止识别您的计算机。请参阅 Unity 知识库文章：我刚刚更新/安装了一个新的操作系统。为什么我的许可证无法激活？(I Have Just Updated/Installed A New Operating System.Why Is My License Failing To Activate?)
  • Unity 的许可证服务器可能已宕机。请参阅 Unity Cloud System Status Page 以查看所有 Unity 服务器的最新状态。

有关所有潜在错误以及如何解决这些错误的信息，请参阅 Unity 支持知识库。如果找不到解决方案，请联系 Unity 客户服务部以获得更多帮助。

__注意：__如果在此页面上留下反馈，我们的支持团队不会收到通知。如果在此过程中遇到任何问题，请向 Unity 客户服务部发送支持工单。
