内置包

内置包允许用户通过 Package Manager 启用或禁用 Unity 功能。启用或禁用某个包会减小运行时构建大小。例如，大多数项目不使用旧版粒子系统。

删除此功能的内置包时，Unity 不会在您构建最终应用程序时包含相关的代码和资源。通常，这些内置包中仅包含包清单，并与 Unity 捆绑在一起（而不是在包注册表中提供）。
