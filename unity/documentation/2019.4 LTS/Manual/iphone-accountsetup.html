iOS 帐户设置

获得 Apple ID 或 Apple Developer Program 会员资格

要在 iOS 设备上测试您的构建版本，您需要一个免费的 Apple ID。但是，要使用诸如游戏中心 (Game Center) 和应用内购 (In-App Purchases) 之类的服务并将游戏提交到 App Store，您必须注册 Apple Developer Program。有关 Apple ID 和 Apple Developer Program 会员资格之间的区别的更多信息，请参阅 Apple 支持文档中的选择会员资格 (Choosing a Membership)。

注册方法： * 要注册 Apple ID，请访问 Apple ID 网站。 * 要注册 Apple Developer Program 成员资格，请访问

安装 Xcode

Xcode 是一个由 Apple 开发且仅适用于 Mac 的集成开发环境 (IDE)，包含一套软件开发工具。您可以使用 Xcode 开发适用于 macOS、iOS、WatchOS 和 tvOS 的软件。

从 App Store 下载 Xcode。

将 Apple ID 添加到 Xcode

要将您的 Apple ID 添加到 Xcode，请按照 Apple 文档中的将 Apple ID 帐户添加到 Xcode (Adding an Apple ID account to Xcode) 部分介绍的步骤进行操作。

  • 于 2019–09–19 更新了 Apple 最新 Xcode、Apple ID 和 Apple Developer Program 文档的链接。
