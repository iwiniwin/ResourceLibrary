串流控制器 (Streaming Controller)

Streaming Controller 组件是纹理串流系统的组成部分。如果要在运行时使用纹理串流将游戏对象的 Mipmap 加载到 Unity 中，请为游戏对象添加 Streaming Controller 组件。

如需了解关于如何使用 Streaming Controller 组件的更多详细信息，请参阅纹理串流 API：控制用于纹理串流的摄像机文档页面

属性 描述
Mip Map Bias 使用此设置可以强制 Unity 加载比纹理串流系统为这些纹理选择的更高或更低的 Mipmap 级别。使用数字字段来设置 Unity 应用于 Mipmap 级别的偏移。Unity 可将此偏移添加到在此摄像机中可见的所有纹理。

要通过 API 来控制此设置，请使用 StreamingController.streamingMipmapBias。
