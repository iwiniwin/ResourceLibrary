将帐户关联到 Google Analytics

Google Analytics 是一个基于 Web 的信息中心，用来显示网站和网页的统计信息。作为发布者，可能会看到 Asset Store Publisher Administration 部分的 Info 选项卡下的 Tracking ID 文本字段。

1.转到 Google Analytics 主页，然后用您的 Gmail 帐户登录。

在 Google Analytics 中，"新帐户"是指要跟踪的新链接或 URL。首次登录 Google Analytics 时，该网站会提示您创建一个新帐户。

2.要创建新帐户，请单击 Start for free 按钮，然后单击 Start measuring 按钮。

此时会显示 Google Analytics **Create account** 窗体。

3.在 Account name 文本框中输入资源包的 Short URL。在 Asset Store Publisher 门户网站的 Packages 选项卡中的 Public Links 部分下可以找到该 Short URL：

![**Public Links** 部分显示了资源页面的长 URL 和短 URL](../uploads/Main/AssetStoreAnalytics.png)

4.按照其余说明设置您的 Google Analytics 帐户。帐户设置完成后，有一个管理员页面会显示有关帐户的信息，包括新帐户的 Tracking ID。

5.复制这个 Tracking ID 并导航到 Asset Store Publisher 门户的 Info 选项卡。

6.在 Info 选项卡上的 Google Analytics 部分，将该 ID 粘贴到 Tracking ID 文本框中。

![Tracking ID 字段是必填字段，而 Location Prefix 字段是选填字段](../uploads/Main/AssetStoreAnalytics-trackingID.png)

7.（可选）还可以在 Location Prefix 框中输入一个值。

8.单击 Save and Preview 按钮以完成操作。

有关 Google Analytics 的更多信息，请参阅 Google 的 Marketing Platform 网站。
