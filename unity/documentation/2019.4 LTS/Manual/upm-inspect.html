检查包

Project 视图会列出项目中当前从所有来源安装的包。这意味着从包注册表安装的不可变包以及可变包（例如嵌入式包和本地包）都是可见的。

您可以检查在 Project 视图中显示的任何包的内容。也可以通过专用的 Inspector 检查包清单。

要检查某个包清单，请在 Project 视图中单击此包清单。

对于嵌入式或本地包，您可以更改包内容以及编辑包清单。
