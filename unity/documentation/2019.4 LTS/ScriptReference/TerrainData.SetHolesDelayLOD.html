TerrainData.SetHolesDelayLOD

切换到手册
public void SetHolesDelayLOD (int xBase, int yBase, bool[,] holes);

参数

xBase要设置的地形孔洞样本的第一个 x 索引。
yBase要设置的地形孔洞样本的第一个 y 索引。
holes要设置的地形孔洞样本的数组（数组以 [y,x] 的形式进行索引）。

描述

设置地形孔洞样本数组。

使用一个二维的地形孔洞样本数组设置地形孔洞数据。样本表示为布尔值：true 表示表面，false 表示孔洞。该数组维度定义受影响的区域（从 xBase 和 yBase 开始）。地形孔洞数组以 [y,x] 的形式进行索引。

与 TerrainData.SetHoles 不同，该方法不更新地形或任何树木/植被对象的 LOD 信息；这意味着某些树/植被对象仍可能存在于孔洞上，但可使该方法快到足以在交互式编辑场景中使用。完成对地形的修改后（例如，当用户释放鼠标按钮时），调用 TerrainData.SyncTexture 并将 TerrainData.HolesTextureName 用作纹理名称来更新所有 LOD 和植被信息。
