ContactFilter2D.IsFilteringDepth

切换到手册
public bool IsFilteringDepth (GameObject obj);

参数

obj GameObject，用于检查 Transform.position 的 z 位置（深度）。

返回

bool 如果该筛选器不包括 /obj/，返回 true，否则返回 false。

描述

检查 obj 的 Transform 是否在要筛选的深度范围内。

筛选被定义为根据指定的条件包括或排除对象。深度筛选对 GameObject Transform.position 的 z 位置进行检查，并当其在深度范围内时包含它，否则排除。 在将 useDepth 设置为 true 并且 obj 变换对象的 z 位置不在 minDepth 和 maxDepth 定义的深度范围内时，IsFilteringDepth 返回 true。这表示该 obj 被筛选掉，这意味着应将其排除。否则 IsFilteringDepth 返回 false。 注意：将 useOutsideDepth 设置为 true 会反转此函数行为，并且返回相反的结果。 另请参阅：useDepth、::ref:minDepth 和 maxDepth。
