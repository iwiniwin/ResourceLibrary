TextureFormat

enumeration

切换到手册

描述

通过脚本创建纹理时使用的格式。

using UnityEngine;

public class Example : MonoBehaviour
{
    void Start()
    {
        // Create a new alpha-only texture and assign it
        // to the renderer's material
        Texture2D texture = new Texture2D(128, 128, TextureFormat.Alpha8, false);
        GetComponent<Renderer>().material.mainTexture = texture;
    }
}

注意，并非所有显卡都支持所有纹理格式。请使用 SystemInfo.SupportsTextureFormat 进行检查。

另请参阅：Texture2D、texture assets。
