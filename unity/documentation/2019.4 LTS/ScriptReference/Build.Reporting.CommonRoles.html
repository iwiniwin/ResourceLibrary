CommonRoles

class in UnityEditor.Build.Reporting

切换到手册

描述

该类为构建中的文件使用的一些常见角色提供常量值。构建中的每个文件的角色可在 BuildFile.role 中找到。

静态变量

appInfo提供在 Windows 低完整性模式下使用的配置信息的文件的 BuildFile.role 值。
assetBundle构建的 AssetBundle 文件的 BuildFile.role 值。
assetBundleTextManifestAssetBundle 清单文件的 BuildFile.role 值，该文件在构建过程中生成且包含有关捆绑包及其依赖关系的信息。
bootConfig包含引擎启动早期阶段的配置信息的文件的 BuildFile.role 值。
builtInResources包含引擎内置资源的文件的 BuildFile.role 值。
builtInShaders包含标准着色器等 Unity 内置着色器的文件的 BuildFile.role 值。
crashHandler用于捕获播放器崩溃情况的可执行文件的 BuildFile.role 值。
debugInfo包含调试器信息的文件的 BuildFile.role 值。
dependentManagedLibrary托管库的 BuildFile.role 值，该库由于是 CommonRoles.managedLibrary 的依赖项而存在于构建中。
engineLibrary当主 Unity 运行时作为单独的库构建时的 BuildFile.role 值。
executable可执行文件的 BuildFile.role 值 - 将在目标设备上实际启动的文件。
globalGameManagers包含播放器的全局项目设置数据的文件的 BuildFile.role 值。
managedEngineApi为 Unity 提供托管 API 的文件的 BuildFile.role 值。
managedLibrary包含编译脚本代码的托管程序集的 BuildFile.role 值。
manifestAssetBundle清单 AssetBundle 的 BuildFile.role 值，该 AssetBundle 包含有关其他 AssetBundle 及其依赖关系的信息。
monoConfig由 Mono 运行时用作配置数据的文件的 BuildFile.role 值。
monoRuntime构成 Mono 运行时的文件的 BuildFile.role 值。
resourcesFile包含项目“资源”文件夹中内容（打包到单个文件中）的文件的 BuildFile.role 值。
scene包含场景打包内容的文件的 BuildFile.role 值。
sharedAssets包含场景之间共享的资源对象的文件的 BuildFile.role 值。资源对象示例包括纹理、模型和音频。
streamingAsset在未修改项目中 StreamingAssets 文件夹的情况下复制到构建中的文件的 BuildFile.role 值。
streamingResourceFile包含流媒体资源数据的文件的 BuildFile.role 值。
