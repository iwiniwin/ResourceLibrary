VideoPlayer.GetAudioLanguageCode

切换到手册
public string GetAudioLanguageCode (ushort trackIndex);

参数

trackIndex要查询的音频轨道的索引。

返回

string 语言代码。

描述

返回指定轨道的语言代码（如果有）。

对于 URL 源，只有在源准备完成后才会设置。请参阅 VideoPlayer.Prepare。
