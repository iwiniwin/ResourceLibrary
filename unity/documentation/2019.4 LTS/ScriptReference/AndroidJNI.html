AndroidJNI

class in UnityEngine

切换到手册

描述

用于来自 Unity 脚本 (C#) 的 Android Java VM 的“原始”JNI 接口。

注意：必须具备 Android Java 原生接口 (JNI) 的高级知识才能使用原生 JNI 函数。请做好记录。
