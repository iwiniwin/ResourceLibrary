PackedAssets

class in UnityEditor.Build.Reporting

/

继承自：Object

切换到手册

描述

BuildReport 类的扩展，用于跟踪资源对构建大小的影响。

构建过程为每个 .sharedAssets 或 .resource 文件或每个 AssetBundle 生成一个 PackedAssets 报告。

变量

contents PackedAssetInfo 对象的数组，该对象保存着 PackedAssets 捆绑包中包含的资源的相关信息，例如打包资源的大小和类型。
file保存着打包资源文件内容的 BuildFile 的标识符。
overhead打包资源文件的标头大小。
shortPath资源包的文件路径（相对于构建的 Data 文件夹）。

继承的成员

变量

hideFlags该对象应该隐藏、随场景一起保存还是由用户修改？
name对象的名称。

公共函数

GetInstanceID返回对象的实例 ID。
ToString返回对象的名称。

静态函数

Destroy移除 GameObject、组件或资源。
DestroyImmediate立即销毁对象 /obj/。强烈建议您改用 Destroy。
DontDestroyOnLoad在加载新的 Scene 时，请勿销毁 Object。
FindObjectOfType返回第一个类型为 type 的已加载的激活对象。
FindObjectsOfType返回所有类型为 type 的已加载的激活对象的列表。
Instantiate克隆 original 对象并返回克隆对象。
