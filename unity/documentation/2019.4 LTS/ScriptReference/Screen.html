Screen

class in UnityEngine

切换到手册

描述

访问显示信息。

Screen 类可用于获取支持的分辨率列表、切换当前分辨率、 隐藏或显示系统鼠标指针。

静态变量

autorotateToLandscapeLeft是否允许自动旋转到左侧横向模式？
autorotateToLandscapeRight是否允许自动旋转到右侧横向模式？
autorotateToPortrait是否允许自动旋转到纵向模式？
autorotateToPortraitUpsideDown是否允许自动旋转到颠倒纵向模式？
brightness当前的屏幕亮度。
currentResolution当前屏幕分辨率（只读）。
cutouts返回不可用于显示内容 (Read Only) 的屏幕区域的列表。
dpi屏幕/设备的当前 DPI（只读）。
fullScreen游戏是否以全屏模式运行？
fullScreenMode将该属性设置为 FullScreenMode 中的某个值，以更改应用程序的显示模式。
height屏幕窗口的当前高度（以像素为单位）（只读）。
orientation指定屏幕的逻辑方向。
resolutions显示器支持的所有全屏分辨率（只读）。
safeArea返回屏幕的安全区域（以像素为单位）（只读）。
sleepTimeout节能设置，允许屏幕在无用户交互一段时间后变暗。
width屏幕窗口的当前宽度（以像素为单位）（只读）。

静态函数

SetResolution切换屏幕分辨率。
