RenderSettings.ambientProbe

切换到手册
public static Rendering.SphericalHarmonicsL2 ambientProbe ;

描述

自定义或天空盒环境光照数据。

天空盒环境光照模式使用此球谐函数 (SH) 探针来计算环境。您也可通过这种方式分配一个完全自定义的 SH 探针。

GI 系统将烘焙环境探针，但实际上该探针不会用在使用光照探针或光照贴图的模型上，因为光照探针和光照贴图中已经存在环境光照了。如果某个对象的光照探针或光照贴图不存在或未启用，系统会将它用作最终的回退项。

调整环境探针不会影响实时和烘焙全局光照的输入。如果您想以影响 GI 的方式调整环境，可通过 RenderSettings.ambientMode 调整环境，例如使用 AmbientMode.Trilight。GI 系统会将产生的环境值输出到环境探针中，这意味着 GI 系统将覆盖自定义环境探针。

另请参阅：ambientMode、SphericalHarmonicsL2、Lighting Window。
