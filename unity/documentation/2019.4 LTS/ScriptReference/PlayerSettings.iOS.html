静态变量

allowHTTPDownload是否应允许不安全的 HTTP 下载？
appInBackgroundBehavior进入后台后的应用程序行为。
appleDeveloperTeamID使用您的 Apple 开发者团队 ID 设置此属性。您可以在 Apple 开发者网站上的 Account > Membership 下找到此信息。这将为生成的 Xcode 项目设置团队 ID，从而允许开发者使用 Build and Run 功能。必须在此处设置 Apple 开发者团队 ID，以自动签署应用程序。
appleEnableAutomaticSigning将此属性设置为 true，以便 Xcode 尝试根据您的 appleDeveloperTeamID 自动签署应用程序。
applicationDisplayNameiOS 应用程序显示名称。
backgroundModes支持的后台执行模式（当 appInBackgroundBehavior 设置为 Custom 时）。
buildNumber捆绑包的版本号
cameraUsageDescription说明访问用户摄像机的原因。
deferSystemGesturesMode延迟系统手势，直到第二次滑动特定边缘。
disableDepthAndStencilBuffers禁用深度和模板缓冲区。
forceHardShadowsOnMetal在（移动端）Metal 上运行时是否应强制执行硬阴影。
hideHomeButton指定是否应在此应用程序的 iOS 版本中隐藏主屏幕按钮。
iOSManualProvisioningProfileID一种资源调配配置文件通用唯一标识符 (UUID)，Xcode 会使用该标识符在手动签署模式下构建您的 iOS 应用程序。
iOSManualProvisioningProfileType在构建 iOS Xcode 项目时要设置的 ProvisioningProfileType。
iOSUrlSchemes应用程序支持的 URL 方案的数组。
locationUsageDescription说明访问用户位置数据的原因。
microphoneUsageDescription说明访问用户麦克风的原因。
prerenderedIcon预渲染图标。
requiresFullScreenRequiresFullScreen 与 Apple 的 plist 构建设置 UIRequiresFullScreen 对应，后者用于选择不参与 Slide Over 和 Split View（适用于 iOS 9.0 多任务处理）。
requiresPersistentWiFi应用程序需要持久性 WiFi。
scriptCallOptimization用于调用优化的脚本。
sdkVersion用于构建的活动 iOS SDK 版本。
showActivityIndicatorOnLoading应用程序应在加载时显示 ActivityIndicator。
statusBarStyle状态栏样式。
targetDevice目标设备。
targetOSVersionStringiOS 的部署最低版本。
tvOSManualProvisioningProfileID一种资源调配配置文件通用唯一标识符 (UUID)，Xcode 会使用该标识符在手动签署模式下构建您的 tvOS 应用程序。
tvOSManualProvisioningProfileType在构建 tvOS Xcode 项目时要设置的 ProvisioningProfileType。
useOnDemandResources指示应用程序是否使用按需加载资源 (ODR) API。

静态函数

SetiPadLaunchScreenType设置将用于为 iPad 生成应用程序启动屏幕界面构建器 (.xib) 文件的模式。
SetiPhoneLaunchScreenType设置将用于为 iPhone 生成应用程序启动屏幕界面构建器 (.xib) 文件的模式。
SetLaunchScreenImage为指定的 iOS 设备设置在游戏启动时要在屏幕上显示的图像。
