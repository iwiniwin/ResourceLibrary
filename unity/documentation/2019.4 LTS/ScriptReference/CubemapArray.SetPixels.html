CubemapArray.SetPixels

切换到手册
public void SetPixels (Color[] colors, CubemapFace face, int arrayElement, int miplevel);

参数

colors像素颜色的数组。
face将为其设置像素的立方体贴图面。
arrayElement将为其设置像素的数组元素索引。
miplevel将为其设置像素的 Mipmap 级别。

描述

为单个数组切片/面设置像素颜色。

此函数采用颜色数组，并且更改一个立方体贴图面的整个 Mip 级别的 像素颜色。调用 Apply 可实际将更改的像素 上传到显卡。

另请参阅：SetPixels32、SetPixelData、GetPixels、Texture2D.SetPixels、Graphics.CopyTexture。
