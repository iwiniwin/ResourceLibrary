MultipartFormDataSectionConstructor

切换到手册
public MultipartFormDataSection (byte[] data);

参数

data此节的数据有效负载。

描述

原始数据节，未命名且没有 Content-Type 标头。

不会包含文件名或 Content-Type 节标头。

public MultipartFormDataSection (string name, byte[] data);

参数

name节名称。
data此节的数据有效负载。

描述

原始数据节，具有节名称，但没有 Content-Type 标头。

与上一个构造函数相同，但包含节名称。

public MultipartFormDataSection (string name, byte[] data, string contentType);

参数

name节名称。
data此节的数据有效负载。
contentType此节的 Content-Type 标头的值。

描述

原始数据节，具有节名称和 Content-Type 标头。

public MultipartFormDataSection (string name, string data, Encoding encoding, string contentType);

参数

name节名称。
data此节的字符串数据有效负载。
contentType此节的 Content-Type 标头的值。
encoding将编组 data 编码为原始字节或者反向操作。

描述

已命名的原始数据节，其有效负载派生自字符串，具有 Content-Type 标头。

使用 encoding 将 data 编码为原始字节。

public MultipartFormDataSection (string name, string data, string contentType);

参数

name节名称。
data此节的字符串数据有效负载。
contentTypeC。

描述

已命名的原始数据节，其有效负载派生自 UTF8 字符串，具有 Content-Type 标头。

对于具有自定义 Content-Type 标头的 UTF8 字符串，请使用此构造函数。假定 data 采用的是 UTF8 编码形式并相应地将其转换为原始字节。

public MultipartFormDataSection (string name, string data);

参数

name节名称。
data此节的字符串数据有效负载。

描述

已命名的原始数据节，其有效负载派生自 UTF8 字符串，具有默认的 Content-Type。

对于 UTF8 字符串，请使用此构造函数。假定 data 采用的是 UTF8 编码形式并相应地进行转换。将为该节分配 text/plain; encoding=utf8 形式的 Content-Type。

public MultipartFormDataSection (string data);

参数

data此节的字符串数据有效负载。

描述

匿名原始数据节，其有效负载派生自 UTF8 字符串，具有默认的 Content-Type。

与上一个构造函数相同，但没有节名称。
